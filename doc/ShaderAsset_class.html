ShaderAsset

Class

Inherits

RBAsset

Description

Contains a shader asset. Shader assets can be loaded from various sources, including synchronous and asynchronous sources. Use ShaderAsset.Load to load a shader asset.

Variables

progress (inherited) Asynchronous loading progress, ranges from 0.0 to 1.0.
shader RetroBlit shader wrapper for this shader

Properties

error (inherited) Loading error of the audio asset
status (inherited) Loading status of the an asset
