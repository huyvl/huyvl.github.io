TMXMapAsset::TMXObjectGroup

Class

Description

A definition of a TMX object group. Object groups are loaded and stored in a TMXMapAsset.

Properties

alpha Alpha transparency of the objects in this group.
color Color used for objects in this group
name Name of the object group
objects List of all objects in this group
offset Offset of the object group in pixels.
properties Custom properties for this object group
visible Visibility flag.
